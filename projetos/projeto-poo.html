Programação Orientada a

Projeto académico focado em conceitos de programação orientada a objetos

Visão Geral

Projeto desenvolvido no âmbito da unidade curricular de Programação Orientada a Objetos. O objetivo foi aplicar os princípios fundamentais da programação orientada a objetos na construção de uma aplicação estruturada e modular.

Durante o desenvolvimento foram utilizadas classes, objetos e diferentes mecanismos de abstração para organizar o código e melhorar a reutilização e manutenção da aplicação.
